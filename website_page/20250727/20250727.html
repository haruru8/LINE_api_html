■ Reactって何？

なんかサマーインターンの申し込みをする際にReactの使用経験とかよく聞かれるけど、なんなんだろうって思って、調べてみた。

ReactはUIを作るためのJavascriptのライブラリらしい

詳しくはReact公式ドキュメントで確認できる。

ってことはReactを学ぶにはまずはJavascriptを勉強する必要があるってことかー

■ とりあえずReactを使えるようにしてみる

Python仮想環境を作成:

python3 -m venv venv

Viteを使ってReactプロジェクトを作成:

npm create vite@latest

実行するとこんな感じで出てくるから下のnow run に書いてある以下を実行

cd vite-project
npm install
npm run dev

これをすることで http://localhost:5173/ を起動できる

■ Reactを勉強してみる

基本的にReact公式ドキュメントを利用して勉強してみる

今日はさわりの部分だけでもやってみた

1. コンポーネントの基本

Reactはコンポーネントという部品でアプリを作る。コンポーネントは関数で作れる。

function MyButton() {
  return (
    <button>I'm a button</button>
  );
}

→ ボタンを作る関数。HTMLっぽく書けるのがJSXという記法

2. コンポーネントを組み合わせる

作ったコンポーネントは他のコンポーネントの中で使える。

export default function MyApp() {
  return (
    <div>
      <h1>Welcome to my app</h1>
      <MyButton />
    </div>
  );
}

→ <MyButton />で先ほど作ったボタンを表示。大文字で始める必要がある。

→ <export default 関数名 { >出力する箇所は多分これ　関数名のところでコードのように関数を定義することもできる

3. データを表示する

JavaScript の変数を {} で囲むとHTMLの中に表示できる。

const user = {
  name: 'Hedy Lamarr',
  imageUrl: 'https://i.imgur.com/yXOvdOSs.jpg'
};

function Profile() {
  return (
    <>
      <h1>{user.name}</h1>
      <img src={user.imageUrl} alt="Photo" />
    </>
  );
}

→ {user.name}で変数の中身を表示。<></>は複数の要素をまとめる時に使う。

4. 条件分岐

条件によって表示を変えたい時は三項演算子を使う。

<div>
  {isLoggedIn ? (
    <AdminPanel />
  ) : (
    <LoginForm />
  )}
</div>

→ isLoggedInがtrueなら管理画面、falseならログイン画面を表示

5. リストの表示

配列のデータを一覧表示する時はmap()を使う。

const products = [
  { title: 'Cabbage', id: 1 },
  { title: 'Garlic', id: 2 },
  { title: 'Apple', id: 3 }
];

const listItems = products.map(product =>
  <li key={product.id}>
    {product.title}
  </li>
);

→ 配列の各アイテムを<li>に変換。keyは必須。

6. イベント処理（クリックなど）

ボタンがクリックされた時の処理を書く。

function MyButton() {
  function handleClick() {
    alert('You clicked me!');
  }

  return (
    <button onClick={handleClick}>
      Click me
    </button>
  );
}

→ onClick={handleClick}でクリック時の関数を指定。括弧は付けない。

7. State（状態管理）

画面の数値を変更したい時はuseStateを使う。

import { useState } from 'react';

function MyButton() {
  const [count, setCount] = useState(0);

  function handleClick() {
    setCount(count + 1);
  }

  return (
    <button onClick={handleClick}>
      Clicked {count} times
    </button>
  );
}

→ countが現在の値、setCountで値を変更。クリックするたびに数字が増える。
